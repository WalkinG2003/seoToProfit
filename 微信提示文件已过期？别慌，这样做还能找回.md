:pushpin: 拯救微信过期文件急救攻略（亲测有效）
很多人都会遇到这种情况：
:backhand_index_pointing_right: 微信里收到的 照片 / 文件 忘记及时接收
:backhand_index_pointing_right: 再点开时提示 “文件已过期”
:backhand_index_pointing_right: 以为彻底没救了

其实在很多情况下，文件并没有真正消失，只是被微信隐藏了。
下面教你 2 种常见、有效的找回方法。

方法一：修改手机日期，重新接收过期照片（适用于未查看的文件）
:pushpin: 适用场景：

照片 / 文件 从未点开查看
只是因为超过 7 天提示过期
操作步骤：
:one: 打开手机设置
:two: 将手机日期 调整到接收文件后的 7 天内
:three: 返回微信
:four: 重新点击之前提示过期的照片
:five: 即可重新打开并保存

方法二：通过微信存储空间找回过期文件（重点推荐）
:pushpin: 适用场景：

文件已经点开过
提示“文件已过期 / 已失效”
且你 没有清理过聊天记录
操作路径：
:one: 打开微信
:two: 点击【我】 → 【设置】
:three: 进入【通用】
:four: 点击【存储空间】
:five: 选择【聊天记录管理】
:six: 找到对应好友的聊天对话框
:seven: 进入后即可看到 过期的图片 / 文档 / 文件

:pushpin: 重点说明：

只要你没有手动清理聊天记录
不管是 图片、视频还是文档
都有很大概率可以在这里找到并重新保存
最后总结（一定要看）
:white_check_mark: 微信提示“文件已过期”
:cross_mark: 不代表文件已经彻底没了

你可以依次尝试：

:one: 先改日期（适合未查看的文件）
:two: 再查存储空间 → 聊天记录管理（成功率最高）

:backhand_index_pointing_right: 只要聊天记录还在，大多数文件都能救回来。
